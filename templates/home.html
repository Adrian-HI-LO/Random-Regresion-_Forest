Sobre el Proyecto

Este proyecto implementa un sistema de detección de malware en Android utilizando técnicas de Machine Learning. El análisis se basa en el dataset CICAAGM que contiene características del tráfico de red de aplicaciones Android. El servidor procesa los datos de forma independiente y entrena dos modelos:

  • Random Forest Classifier: Para clasificar aplicaciones como benignas o maliciosas
  • Random Forest Regressor: Para predecir la duración del tráfico de red
